## Тестирование Itunes для Windows 10
## Установка
* 1 Открыть Microsoft Store
* 2 Нажать на поиск
* 3 Набрать Itunes
* 4 Перейти на страницу приложения (Itunes)
* 5 Кликнуть на кнопку загрузить
* 6 Ждать
